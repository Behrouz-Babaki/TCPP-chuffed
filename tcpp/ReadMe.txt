How to Run:

TCPP Domain.pddl problem.pddl -varorder conflict -preprocess SAC -prop-node SAC AuxPrefixName


Domain.pddl and problem.pddl are domain and problem files of a planning problem.

-varorder conflict -preprocess SAC -prop-node SAC are minion solver paremeters(switches).

AuxPrefixName is a name used as prefix of auxilary files and can be any name.
